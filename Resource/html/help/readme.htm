【关于91笔记 V1.3.0】

91笔记是由网龙公司出品的一款快速记事类软件，除了常见的文字记录外，还增加了画图记录、拍摄记录和语音记录等功能，并可以通过账户在网络上进行数据托管，充分发挥智能手机的功能，让用户不受计算机的限制，随时随地用手机访问和编辑日志笔记。

一、软件登录和界面

软件安装完毕后，需要使用91通行证进行登录，当然您也可以点击右软键"随便看看"简单的体会下91笔记的各个功能。二者的区别是使用91通行证登录之后可以进行数据托管，只要存在网络即可随时随地备份和还原文件数据，不必担心因为机器损坏等原因而导致数据丢失。

二、软件功能-随手记

可以通过菜单中的"编辑标题"、"编辑标签"或者直接点击相应的区域进行输入，91笔记目前可以作为一个在线的文档编辑器和修改器使用，只支持文本书写和保存。内容输入完成后点击完成就会自动发送到服务器，上传完成后，可以在主面板中点击左软键"查看笔记"，这个时候就可以看到自己编辑上传的笔记列表。

三、软件功能-随手拍

随手拍功能，可以保存当前的摄像头中获取的图片，如不满意可重拍，如果确认使用则点击使用，软件即可把软件保存至服务器。

四、软件功能-随手画

随手画功能是对随手记功能的一个扩展，这里可以支持彩色涂写功能，笔触有多种尺寸可供选择，色彩也可完全自定义。面板下方按钮分别为：后退、前进、颜色、画笔粗细、相簿、和删除，上方的三个按键为画笔、橡皮擦和移动缩放。

五、软件功能-随手录

随手录功能中，左下角按键为开始录音，中间为播放键，右侧为删除键。当开始录音之后左侧的录音键会变成停止键，再次点击即可停止录音。录音完成后点击右上角的保存即可。

六、网络功能

在使用过程中如果登录91通行证，进行网络数据同步，需有网络支持。产品支持IEEE802.11（WIFI）无线网络和GPRS或3G接入技术，并可以由用户自主选择网络类型。同时产品为用户提供了GPRS流量设置功能，避免因流量超出而造成不必要的损失。
